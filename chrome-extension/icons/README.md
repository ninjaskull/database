# Extension Icons

Place your extension icons here in PNG format:

- `icon16.png` - 16x16 pixels (toolbar)
- `icon48.png` - 48x48 pixels (extensions page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Quick Icon Generation

You can create simple icons using an online tool or use these placeholder SVGs:

```svg
<svg width="128" height="128" viewBox="0 0 128 128" fill="none" xmlns="http://www.w3.org/2000/svg">
  <rect width="128" height="128" rx="24" fill="#3b82f6"/>
  <circle cx="54" cy="54" r="24" stroke="white" stroke-width="8" fill="none"/>
  <line x1="70" y1="70" x2="98" y2="98" stroke="white" stroke-width="8" stroke-linecap="round"/>
</svg>
```

Convert this SVG to PNG at different sizes for each icon file.
